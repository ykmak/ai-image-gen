📖 使用教程

1

获取 API Key
前往 APIYi 注册账号，在控制台获取 API Key

2

填写提示词
描述你想要的画面，越详细效果越好。可使用下方快捷标签快速填充风格

3

上传参考图（可选）
支持拖拽上传，可选择图生图/风格参考/构图参考三种模式，调节参考强度

4

调整参数
选择图片尺寸、质量和数量，进阶用户可展开进阶参数调节采样步数和 CFG

5

点击生成
等待 AI 创作，成品图可下载、复用提示词或以图生图继续创作
